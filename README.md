#### 软件架构
Java + Spring + SpringMVC + Hibernate + Mysql


#### 项目功能说明

1.  管理员功能
> + 个人信息：修改密码
> + 图书管理：添加图书、查看图书、所有借阅
> + 用户管理：用户列表
>   退出登录
2.  普通用户
> + 个人信息
> + 图书归还：归还图书
> + 图书续借：续借图书
> + 图书借阅：图书列表
> + 图书检索：查找图书
> + 退出登录
